# dotfiles

These are my dotfiles.

## install prompt:

1. Symlink (or copy) pure.zsh to ~/.oh-my-zsh/custom/pure.zsh-theme
2. Symlink (or copy) async.zsh to ~/.oh-my-zsh/custom/async.zsh
3. Add ZSH_THEME="pure" to your .zshrc file.
